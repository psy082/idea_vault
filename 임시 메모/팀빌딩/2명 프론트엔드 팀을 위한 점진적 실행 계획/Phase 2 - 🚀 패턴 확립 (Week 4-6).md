### Week 4: 구조화된 멘토링 시작

#### 1:1 멘토링 세션 시작 (주 1회, 45분)

```
세션 구조:

1. 체크인 (10분)
   - 지난주 학습 목표 달성도
   - 이번 주 어려웠던 점
   
2. 기술 토픽 (20분)
   - 파트원이 선택한 주제 깊이 있게 다루기
   - 예: "React의 렌더링 최적화 전략"
   - 파트리더가 화이트보드/코드로 설명
   - 파트원이 코드 예시 만들어보기
   
3. 코드 리뷰 (10분)
   - 이번 주 파트원이 작성한 코드 중 1개 선정
   - 좋은 점 먼저 언급
   - 개선 방향 제시 (비판 X)
   
4. 다음 주 계획 (5분)
   - 다음 멘토링 주제
   - 시도해볼 것
```

#### 코드 리뷰 개선

```
Before (피드백 중심):
"여기서 useMemo를 써야 합니다."

After (멘토링 중심):
"이 컴포넌트가 렌더링되는 상황을 생각해볼까요?
props가 바뀔 때마다 이 계산이 다시 실행되는데,
비용이 큰 계산일까요? 
만약 그렇다면 어떤 최적화를 고려할 수 있을까요?"
```
